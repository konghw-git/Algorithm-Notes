## 2 算法的核心思想

1. 双端队列

```python
from collections import deque

# 1 创建空的双端队列
dq = deque()

# 2 从可迭代对象初始化
dq = deque([1, 2, 3])	# # deque([1, 2, 3])
dq = deque("abc")	# deque(['a', 'b', 'c'])

# 3 设置最大长度 超出时自动移除另一端元素
dq = deque(maxlen=3)

# 4 右侧添加元素
dq.append(1)

# 5 左侧添加元素
dq.appendleft(0)

# 6 批量添加元素
dq.extend([2, 3, 4])
dq.extendleft([1, 2, 3])

# 7 删除元素
dq.pop()	# 移除并返回最右侧元素 队列为空抛出IndexError
dq.popleft() 
dq.remove(2)	# 删除第一个匹配的元素 元素不存在抛出ValueError

# 8 清空队列
dq = deque([1, 2, 3])
dq.clear()	# # deque([])

# 9 查看队列长度
print(len(dq))

# 10 访问元素
dq = deque([1, 2, 3])
print(dq[0])  # 1（第一个元素）
print(dq[-1]) # 3（最后一个元素）

'''
应用场景：
实现栈（用 append() 和 pop()）或队列（用 append() 和 popleft()）
滑动窗口问题（利用 maxlen 自动维护窗口大小）
字符串 / 数组的前后缀操作（如回文判断）
缓存实现（超出 maxlen 时自动淘汰旧元素）
'''
```
